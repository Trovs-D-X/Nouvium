# Welcome to Carl-OS!

# What is Carl-OS?

Carl-OS is our own operating system. It is:

- Free.

- Open source.

- GPL v2.0.

# Why was Carl-OS created?

Here are some reasons:

1. It's unfair to have to pay for customization, using programs, etc.

2. I wanted to allow modifications even if you don't have money.

3. Although I wanted to be free, I didn't want anyone to shut down their modified projects.

4. I wanted to prevent people from creating their own system from scratch with the Carl Kernel.

# Was it built from scratch?

Yes.

CarlOS was made completely from scratch to have total independence from Linux or any other system.

# How to start building your distro using Carl-OS instead of Carl?

1. Download the official versions from this repository.

2. Start modifying whatever you want, but be careful not to break the system. The organization is NOT responsible.

3. Once finished, you are obligated to comply with the license and publish your modified version.

End.
